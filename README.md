# Довідник географа 🌍

Це простий застосунок на C# (WinForms), який дозволяє зберігати та керувати інформацією про різні міста (назва, населення, координати).

## Можливості:
* Додавання нових міст до списку.
* Редагування існуючих записів.
* Видалення міст зі списку.

## Технології:
* C#
* Windows Forms
* .NET
